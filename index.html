Introduction

This API makes use of one API definition path in order to make requests.

Request URL: merlototp.herokuapp.com/


Function Description
Generate OTP Initiate an OTP request using a clientID, thereby generating an OTP and submitting it to the notification subsystem in order to send it to the associated e-mail address.
Validate OTP Validates whether a PIN that is sent in a request matches the saved OTP associated with the clientID.

Generate OTP

Returns the status of whether the OTP was successfully sent.

Parameter Type Constraint
clientID string Must be an existing clientID in the Client Information System
Type string "generate"
Example Request: {
  "clientID":"003",
  "type":"generate"
}
Successful
Example Response: {
  "ClientID":"003",
  "Success":"true"
}
Failed
Example Response: {
  "ClientID":"003",
  "Success":"true"
  "message":"Something went wrong"
}

Validate OTP

Returns the status of whether the OTP - that had been previously generated - and PIN - that has been sent in the request - match.

Parameter Type Constraint
clientID string Must be an existing clientID in the Client Information System
Type string "validate"
pin string 4 digits
Example Request: {
  "clientID":"003",
  "type":"validate"
  "pin":"4567"
}
Successful
Example Response: {
  "ClientID":"003",
  "Success":"true"
  "message":"Correct OTP"
}
Failed
Example Response: {
  "ClientID":"003",
  "Success":"true"
  "message":"Incorrect OTP or no OTP created"
}
